## 為什麼要成為一個數據高速公路授權機構？

初始的 Data高速公路測試網將使用授權證明 (PoA) 共識，在這種共識中，驗證器 (授權) **用其身份 **來驗證交易和區塊以確保區塊鏈的安全。

符合資格的通證或硬件挖礦者聲稱的挖礦速度提升乘數可以用來獲得更高的區塊獎勵
